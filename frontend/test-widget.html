Chatbot Widget Test Page

Instructions:

This page demonstrates how to embed the chatbot widget on your website.

The chatbot widget should appear in the bottom-right corner of the page.

Replace YOUR_CHATBOT_ID with your actual chatbot ID from the dashboard.

Embedding Code:

<script src="https://chatbotfrontend-gamma.vercel.app/widget.js"></script>
<script>
  const widget = new ChatbotWidget({
    chatbotId: 'YOUR_CHATBOT_ID',
    position: 'bottom-right', // or 'bottom-left'
    primaryColor: '#3b82f6' // optional: customize color
  });
  widget.init();
</script>
